Question-5 Explain CSS box model

It consists of: content, padding, borders and margins.
Content - The content of the box or text or images in the box
Padding - An area around the content.
Border - A border around the padding and content.
Margin - An area outside the border.

This is CSS Box Model

Question-6 what is the difference between display none and visibility hidden?

"display: none":It completely removes the element from the webpage
"visibility:hidden": It only hides the element but keeps its position intact.
